# Cactus Clicker

**Cactus Clicker** is a casual clicking game developed in **C++** using **SFML**

## Features

- Sound effects and background music
- Clean and modular C++ code

## Tools Used

- **Framework/Engine:** [SFML](https://www.sfml-dev.org/)
- **Language:** [C++](https://isocpp.org/std/the-standard)
- **Development environment:** [Visual Studio](https://visualstudio.microsoft.com/es/)
- **Image creation and editing:** [Canva](https://www.canva.com/)
